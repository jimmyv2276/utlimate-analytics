# Ulti Analytics

A personal learning project inspired by tools like UltiAnalytics.

The goal of this project is to practice:
- Event-driven data modeling
- Sports analytics and statistics
- Mobile-first data entry workflows
- Backend API and infrastructure best practices
- Gradual integration of AI-assisted features

This project is intentionally built incrementally, focusing on correctness,
clarity, and learning over speed.

---

## Tech Stack (initial)

- **Backend**: Python, FastAPI
- **Database**: PostgreSQL (via Docker)
- **Infra**: Docker, Docker Compose
- **Frontend**: TBD (mobile-first, likely React)
- **Analytics**: SQL + Python (pandas, etc.)
